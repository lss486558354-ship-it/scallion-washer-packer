# 继电保护设计

大葱切割-清洗-打包一体机 — 电气保护与控制逻辑

> 版本：V2.0 | 更新日期：2026-04-03 | 适用范围：STM32F103VE
> **电源方案：4S~6S 锂电池（14.8V~22.2V）→ 升压板 → DC 24V / DC 5V**

---

## 一、设计目标

1. **双重保险**：急停按钮 + 单片机继电器控制，任何一个都能切断主电路
2. **自锁回路**：按下启动按钮后继电器自锁，保证持续供电
3. **上电安全**：单片机控制继电器确保逻辑上电后才给设备供电
4. **急停优先**：急停按钮直接断开主电路（物理上优先于软件）
5. **电池保护**：升压板前有过流/欠压保护，延长电池寿命

## 四、控制逻辑详解

### 4.1 上电时序

```
电池接入（14.8V~22.2V）
  │
  ├─→ 升压板工作 → 输出24V / 5V（升压板自启动）
  │
  ├─→ DC 5V → STM32上电
  │
  └─→ DC 24V → 等待K_M主触点闭合

STM32启动：
  1. 初始化 → PA15输出低 → K_S继电器断开（安全默认）
  2. PE6检测急停按钮状态
  3. 等待启动

用户按SB_ON：
  1. K线圈得电 → K_M主触点闭合 → 升压板输入得电
     （即使K_S断开，升压板已在工作，K_M合上后设备得电）
  2. K辅助常开闭合 → 自锁保持
  3. PA15输出高 → K_S闭合 → 双重保险

运行时：
  K_M主触点 ──→ 闭合（自锁）
  K_S继电器 ──→ 闭合（单片机控制）
  设备供电 ────→ 正常
```

### 4.2 停止时序

```
正常停止（按SB_OFF）：
  SB_OFF常闭断开 → K线圈断电
  → K_M主触点断开 → 升压板输入断开
  → K辅助常开断开 → 自锁解除
  → PE6检测到变化
  → PA15输出低 → K_S断开

单片机控制停止（Process_Stop）：
  PA15输出低 → K_S断开
  → K线圈断电（如果K_S与K线圈串联）
  → K_M主触点断开 → 全系统断电
```

### 4.3 急停时序（最高优先级）

```
急停按钮按下：
  常闭断开 → K线圈立即断电（硬件物理优先）
  → K_M主触点立即断开
  → 全系统立即断电
  │
  → PE6检测到电平跳变（0→1或1→0）
  → Process_EmergencyStop()
  → PA15输出低 → K_S断开（双重保险）

急停松开：
  → PE6恢复
  → 需按SB_ON重新启动
```

### 4.4 电池电压与保护

| 参数 | 值 | 说明 |
|------|------|------|
| 电池电压（4S）| 14.8V（满电16.8V，放电截止14.8V）| 4S LiPo |
| 电池电压（6S）| 22.2V（满电25.2V，放电截止22.2V）| 6S LiPo |
| 升压板输入范围 | 12V~26V | 需留余量 |
| 升压板24V输出 | 24V / 10A（max）| 驱动步进/直流 |
| 升压板5V输出 | 5V / 3A（max）| STM32/传感器/舵机 |

**建议在升压板前增加：**
- 20A~30A保险丝（短路保护）
- TVS二极管（防浪涌）
- 防反接二极管（可选，防止电池接反）

## 五、单片机控制逻辑（STM32）

### 5.1 继电器控制引脚

| 引脚 | 功能 | 默认状态 | 说明 |
|------|------|---------|------|
| PA15 | K_S 电源控制继电器 | 低电平（断开）| 上电安全默认 |
| PB12 | K_AIR 空压机继电器 | 低电平（断开）| 控制空压机 |
| PB13 | K_PUMP 水泵继电器 | 低电平（断开）| 控制水泵 |
| PE6 | 急停按钮检测 | 上拉输入 | 常闭触发 |

### 5.2 继电器初始化代码

```c
void Relay_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStruct = {0};
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA | RCC_APB2Periph_GPIOB |
                            RCC_APB2Periph_GPIOE, ENABLE);

    GPIO_InitStruct.GPIO_Speed = GPIO_Speed_2MHz;
    GPIO_InitStruct.GPIO_Mode  = GPIO_Mode_Out_PP;

    /* PA15 - 电源控制继电器 K_S（控制升压板前端）*/
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_15;
    GPIO_Init(GPIOA, &GPIO_InitStruct);
    K_S_OFF();  /* 默认断开，安全状态 */

    /* PB12 - 空压机继电器 */
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_12;
    GPIO_Init(GPIOB, &GPIO_InitStruct);
    K_AIR_OFF();

    /* PB13 - 水泵继电器 */
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_13;
    GPIO_Init(GPIOB, &GPIO_InitStruct);
    K_PUMP_OFF();

    /* PE6 - 急停按钮检测（输入，上拉）*/
    GPIO_InitStruct.GPIO_Pin  = GPIO_Pin_6;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_IPU;
    GPIO_Init(GPIOE, &GPIO_InitStruct);
}

/* 急停检测任务（FreeRTOS任务，最高优先级）
 * 使用 vTaskDelayUntil 实现精确的1ms周期检测
 * 所有FreeRTOS任务均使用 vTaskDelayUntil 而非 vTaskDelay 确保精确周期性 */
void vTask_EmergencyStop(void *pvParameters)
{
    TickType_t last_wake_time;
    const TickType_t period = pdMS_TO_TICKS(1);  /* 1ms检测周期 */

    last_wake_time = xTaskGetTickCount();

    uint8_t last_state = 1;
    for (;;) {
        uint8_t curr = GPIO_ReadInputDataBit(GPIOE, GPIO_Pin_6);
        if (last_state == 1 && curr == 0) {
            /* 急停按钮按下（电平由高变低）*/
            Process_EmergencyStop();
            K_S_OFF();    /* 断开K_S */
            K_AIR_OFF();  /* 断开空压机 */
            K_PUMP_OFF(); /* 断开水泵 */
            Device.UART1.SendString("[EMERGENCY STOP] All systems halted!\r\n");
        }
        last_state = curr;
        vTaskDelayUntil(&last_wake_time, period);
    }
}
```

## 六、安全设计要点

### 6.1 硬件优先原则

急停按钮的常闭触点直接串联在继电器K线圈回路中，**硬件断电优先级高于单片机控制**。即使单片机死机，按下急停也能立即切断主电路。

### 6.2 软件安全机制

| 保护措施 | 实现方式 |
|---------|---------|
| 看门狗 | 启用IWDG，防止程序死机 |
| 急停检测 | PE6电平检测，任何时刻都可触发 |
| 状态机异常 | 超时检测，超时自动进入安全状态 |
| 传感器异常 | 传感器数据有效性校验，无效则停机 |
| 继电器安全默认 | 上电默认断开，需程序主动闭合 |
| 电池欠压监测 | 监测电池电压（ADC），低于阈值报警停机 |

### 6.3 接线注意事项

1. **急停按钮**：必须使用常闭型，与继电器线圈直接串联
2. **停止按钮**：常闭型，串在自锁回路中
3. **启动按钮**：常开型，并联在自锁触点旁
4. **K_M主触点**：选型DC 24V/20A以上，继电器或MOSFET模块
5. **K_S继电器**：DC 5V线圈，与STM32共地
6. **保险丝FU**：20A~30A快熔，安装在电池正极输出端
7. **升压板**：确认输入电压范围覆盖4S~6S（12V~25.2V）
8. **共地**：STM32 GND、继电器GND、控制回路GND必须可靠连接
9. **防反接**：建议在电池输出端正向串联二极管（注意压降和功耗）

## 七、故障分析

| 故障现象 | 可能原因 | 排除方法 |
|---------|---------|---------|
| 按启动按钮无反应 | 停止按钮未闭合/接线松脱 | 检查停止按钮和接线 |
| 松开启动按钮后断电 | 自锁触点未接/接触不良 | 检查继电器辅助触点 |
| 单片机无法控制K_S | 继电器线圈接反/STM32未初始化 | 检查接线，检查初始化代码 |
| 急停无法断电 | 急停按钮接成常开 | 改为常闭型接线 |
| 设备意外断电 | 急停按钮抖动/接触不良 | 检查按钮质量 |
| 升压板发热严重 | 负载过重/散热不良 | 检查负载电流，加散热片 |
| 电池很快没电 | 放电电流过大/电池老化 | 检查负载功率，更换电池 |


## 四、控制逻辑

### 4.1 上电时序

```
1. 220V AC → 升压板 → DC 24~48V输出（升压板自启动）
2. DC 5V/3.3V电源模块工作 → STM32上电
3. STM32初始化 → PA15输出低电平 → K_S继电器断开（默认安全状态）
4. 用户按下启动按钮SB_ON
   → 接触器K线圈得电
   → K主触点闭合 → 升压板输入得电（双重得电保证）
   → K辅助常开闭合 → 自锁
5. 同时：K_S继电器在程序正常运行后闭合（双重保险）

上电后状态：
  K 接触器 ──→ 闭合（自锁）
  K_S继电器 ──→ 闭合（单片机控制）
  升压板 ────→ DC输出正常
  设备供电 ──→ 正常
```

### 4.2 停止时序

```
正常停止（按下停止按钮 SB_OFF）：
  SB_OFF 常闭断开 → 接触器K线圈断电
  → K主触点断开 → 升压板断电
  → K辅助常开断开 → 自锁解除
  → 单片机检测到PE6电平变化
  → PA15输出低 → K_S继电器断开（双重保险）

单片机控制停止（Process_Stop）：
  PA15输出低 → K_S继电器断开
  → 接触器K线圈断电（如果K_S与K线圈串联）
  → K主触点断开 → 全系统断电
```

### 4.3 急停时序（最高优先级）

```
急停按钮按下（常开型急停）：
  急停按钮常开闭合 → 继电器K_EST线圈得电
  → K_EST触点断开（与K线圈串联）→ 接触器K断电
  → 主触点全部断开 → 全系统立即断电

急停按钮按下（常闭型急停，本设计采用）：
  急停按钮常闭断开 → 接触器K线圈立即断电（硬件优先）
  → 主触点立即断开 → 全系统立即断电
  → 同时 STM32检测到PE6电平跳变
  → 触发 Process_EmergencyStop()
  → PA15输出低 → K_S继电器断开
```

### 4.4 继电器K_S与接触器K的关系

```
方案A（推荐）：K_S并联在停止按钮两端
  ┌────────────────────────────────────────┐
  │  +24V ─→ SB_OFF(常闭) ──┬──→ 线圈K ──→ GND
  │                        │
  │               K_S常开(单片机控制) ────┘
  └────────────────────────────────────────┘
  → 单片机输出高 → K_S闭合 → 旁路停止按钮 → 接触器保持

方案B：K_S串联在K线圈回路中
  ┌────────────────────────────────────────┐
  │  +24V ─→ SB_OFF ──→ K_S(常开) ─→ 线圈K ──→ GND
  └────────────────────────────────────────┘
  → 单片机输出低 → K_S断开 → K立即失电
```

## 五、单片机控制逻辑（STM32）

### 5.1 继电器控制引脚

| 引脚 | 功能 | 默认状态 | 说明 |
|------|------|---------|------|
| PA15 | K_S 电源控制继电器 | 低电平（断开） | 上电安全默认 |
| PB12 | K_AIR 空压机继电器 | 低电平（断开） | 控制空压机 |
| PB13 | K_PUMP 水泵继电器 | 低电平（断开） | 控制水泵 |

### 5.2 继电器初始化代码

```c
void Relay_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStruct = {0};
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA | RCC_APB2Periph_GPIOB, ENABLE);

    GPIO_InitStruct.GPIO_Speed = GPIO_Speed_2MHz;
    GPIO_InitStruct.GPIO_Mode  = GPIO_Mode_Out_PP;

    /* PA15 - 电源控制继电器 K_S */
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_15;
    GPIO_Init(GPIOA, &GPIO_InitStruct);
    K_S_OFF();  /* 默认断开，安全状态 */

    /* PB12 - 空压机继电器 */
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_12;
    GPIO_Init(GPIOB, &GPIO_InitStruct);
    K_AIR_OFF();

    /* PB13 - 水泵继电器 */
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_13;
    GPIO_Init(GPIOB, &GPIO_InitStruct);
    K_PUMP_OFF();

    /* PE6 - 急停按钮检测（输入，上拉）*/
    GPIO_InitStruct.GPIO_Pin  = GPIO_Pin_6;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_IPU;
    GPIO_Init(GPIOE, &GPIO_InitStruct);
}

/* 急停检测任务（FreeRTOS任务，最高优先级）
 * 使用 vTaskDelayUntil 实现精确的1ms周期检测
 * 所有FreeRTOS任务均使用 vTaskDelayUntil 而非 vTaskDelay 确保精确周期性 */
void vTask_EmergencyStop(void *pvParameters)
{
    TickType_t last_wake_time;
    const TickType_t period = pdMS_TO_TICKS(1);  /* 1ms检测周期 */

    last_wake_time = xTaskGetTickCount();

    uint8_t last_state = 1;
    for (;;) {
        uint8_t curr = GPIO_ReadInputDataBit(GPIOE, GPIO_Pin_6);
        if (last_state == 1 && curr == 0) {
            /* 急停按钮按下（电平由高变低）*/
            Process_EmergencyStop();
            K_S_OFF();
            K_AIR_OFF();
            K_PUMP_OFF();
            Device.UART1.SendString("[EMERGENCY STOP] All systems halted!\r\n");
        }
        last_state = curr;
        vTaskDelayUntil(&last_wake_time, period);
    }
}
```

## 六、安全设计要点

### 6.1 硬件优先原则

急停按钮的常闭触点直接串联在接触器线圈回路中，**硬件断电优先级高于单片机控制**。即使单片机死机，按下急停也能立即切断主电路。

### 6.2 软件安全机制

| 保护措施 | 实现方式 |
|---------|---------|
| 看门狗 | 启用IWDG，防止程序死机 |
| 急停检测 | PE6电平检测，任何时刻都可触发 |
| 状态机异常 | 超时检测，超时自动进入安全状态 |
| 传感器异常 | 传感器数据有效性校验，无效则停机 |
| 继电器安全默认 | 上电默认断开，需程序主动闭合 |

### 6.3 接线注意事项

1. **急停按钮**：必须使用常闭型，与接触器线圈直接串联
2. **停止按钮**：常闭型，串在自锁回路中
3. **启动按钮**：常开型，并联在自锁触点旁
4. **接触器选型**：额定电流≥最大负载电流的1.5倍
5. **熔断器FU**：选型10A，安装在L线进线端
6. **接触器辅助触点**：选型带常开辅助触点用于自锁
7. **继电器K_S**：使用DC线圈继电器，与STM32共地
8. **共地**：STM32GND与控制回路GND必须可靠连接

## 七、故障分析

| 故障现象 | 可能原因 | 排除方法 |
|---------|---------|---------|
| 按启动按钮无反应 | 停止按钮未闭合/接线松脱 | 检查停止按钮和接线 |
| 松开启动按钮后断电 | 自锁触点未接/接触不良 | 检查接触器辅助触点 |
| 单片机无法控制K_S | 继电器线圈接反/STM32未初始化 | 检查接线，检查初始化代码 |
| 急停无法断电 | 急停按钮接成常开 | 改为常闭型接线 |
| 设备意外断电 | 急停按钮抖动/接触不良 | 检查按钮质量 |
